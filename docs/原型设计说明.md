# 设计说明

1) 首页（home.svg）
   - 目的：快速传达项目定位与探索入口。
   - 布局：顶部导航 + Hero（标题/介绍/CTA/地图预览）+ 声景卡片 + 路线/社区/上传入口。
   - 交互：点击“开始探索”进入声景库，地图预览强化“可探索”的核心概念。

2) 声景库列表页（sounds.svg）
   - 目的：高效浏览与筛选。
   - 布局：筛选区（关键词/标签/城市/排序）+ 卡片栅格。
   - 交互：筛选条件即时更新列表；点击卡片进入详情；卡片支持试听。

3) 声音详情页（sound-detail.svg）
   - 目的：完整呈现声音故事与互动信息。
   - 布局：左侧主信息（标题/故事/播放）+ 右侧地图与统计；下方评论与推荐。
   - 交互：播放控制、收藏、评论发布；推荐区跳转同类声音。
