# 实验总结

本次实验围绕“城市声音地图（City Sound Map）”完成了前端页面的静态结构、样式实现与基础交互，并对组件化、路由与响应式设计进行了实践。整体流程包括页面信息架构梳理、HTML5 语义化结构搭建、CSS 布局实现、Vue Router 配置与交互功能落地，产出清晰可提交。

在实现过程中，我重点完成了以下工作：  
1) 使用语义化标签完成多页面 HTML 结构搭建，保证结构清晰可读。  
2) 编写统一的 CSS 样式，综合使用 Flexbox 与 Grid 布局，形成稳定的视觉层级与卡片布局。  
3) 通过媒体查询实现移动端与桌面端的基本自适应。  
4) 配置 Vue Router，实现多页面无刷新切换，并补充交互示例页面。  
5) 封装 CreatorCard 组件，并在首页与社区页复用，提升组件化与维护性。  

遇到的主要问题与解决方式：  
- 路由切换与页面结构需要统一导航入口，通过规范路由配置与页面组织方式解决。  
- 多模块页面在不同屏宽下易出现间距不一致，通过统一的容器间距与断点规则进行修正。  
- 卡片与表单布局在移动端容易拥挤，采用 Grid 单列降级并优化按钮与输入框的间距。  

收获与改进方向：  
本次实验强化了我对语义化结构、布局组合（Flex + Grid）、路由配置与基础交互实现的理解，也提升了组件复用的实践能力。后续可继续完善真实数据接入、交互动画与表单校验，并对移动端细节与性能进行进一步优化。
