## （1）优化措施说明
1) 路由级代码分割（懒加载）
   - 修改 `src/router/index.js`，将路由组件改为动态导入。
   - 原理：将页面组件拆分为独立 chunk，首屏只加载必要代码，减少初始包体积。

2) 图片懒加载与解码优化
   - 在社区话题与图集图片上增加 `loading="lazy"` 与 `decoding="async"`。
   - 原理：延迟加载非首屏图片并异步解码，降低首屏资源压力。
