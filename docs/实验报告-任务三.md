## （1）程序代码（组件代码）
组件名称：CreatorCard  
文件路径：`src/components/CreatorCard.vue`

HTML 结构：
```html
<article class="creator-card">
  <div class="creator-avatar">{{ creator.name.slice(0, 1) }}</div>
  <div>
    <p class="creator-name">{{ creator.name }}</p>
    <p class="creator-role">{{ creator.role }}</p>
    <div class="creator-badges">
      <span v-for="badge in creator.badges" :key="badge">{{ badge }}</span>
    </div>
  </div>
</article>
```

CSS 样式：

## （2）使用方式说明
在页面中引入组件，并传入 `creator` 对象即可渲染创作者卡片。  
`creator` 包含 `name`、`role`、`badges` 字段。

示例用法：
```vue
<script setup>
import CreatorCard from '../components/CreatorCard.vue';

const creators = [
  { name: '余声', role: '城市录音师', badges: ['雨声', '夜市'] }
];
</script>

<template>
  <CreatorCard v-for="creator in creators" :key="creator.name" :creator="creator" />
</template>
```
