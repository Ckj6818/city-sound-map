# 城市声音地图（City Sound Map）需求分析

## 项目名称
城市声音地图（City Sound Map）

## 项目简介
基于 Vue 3 + Vite 的前端项目，通过地图与音频卡片呈现城市声音。用户可以浏览、上传、标注声音位置与时间，生成“声音路线”和故事合集，形成可探索的城市声景。

## 目标用户
- 城市文化爱好者、旅行者，希望用声音记录和探索城市
- 音频创作者/播客主，寻找素材与灵感
- 大学生与摄影/音频社团，用于采集与作品展示

## 功能模块列表（至少 3 个）
1) 地图探索与分类
   - 地图视图展示声音点位，支持缩放与聚合
   - 分类筛选（自然/交通/人声/音乐/环境），支持标签与热度排序
   - 点位详情卡片：音频预览、地点、时间、故事描述

2) 声音上传与标注
   - 上传音频、封面图、标题、描述
   - 地图选点或自动定位，记录时间与环境标签
   - 简易音频裁剪/预览，发布前确认

3) 声音路线与合集
   - 将多个声音点位组成路线（如“雨天街区线”）
   - 路线顺序与地图轨迹展示
   - 一键分享路线海报/链接

4) 社区互动与收藏
   - 点赞、收藏、评论
   - 举报与内容提醒
   - 关注作者，查看其声音作品集

5) 个人中心与数据
   - 我的上传、收藏、路线
   - 播放次数、收藏量、互动通知
   - 资料编辑与头像设置

## 非功能需求（简要）
- 兼容主流浏览器与移动端，自适应布局
- 音频播放流畅，列表懒加载
- 地图加载优化，首屏 < 2s
- 基础内容安全：敏感词过滤与举报流程
